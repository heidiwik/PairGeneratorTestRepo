# PairGenerator

A simple .NET console application that generates random pairs for pair programming assignments.

This repository is for Git and GitHub exercises. See the instructions in the course materials.

## Features

- Reads a list of names from a text file
- Randomly pairs them up for pair programming
- Writes the results back to a file

## Prerequisites

- .NET Core 3.1 SDK

## Usage

1. Create a text file with one name per line
2. Update the file path in `PairGenerator.cs` if needed
3. Run the application:

```
dotnet run --project PairGenerator
```
